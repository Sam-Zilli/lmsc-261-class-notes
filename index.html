Variables

Variables are containers for storing data values.

In this class, we will use the 'let' keyword to declare (make) variables.

You may see code in resources or online that use 'var' or 'const' instead of 'let'

Example:

1.) Declare (make) a variable called x

2.) Bind the value 5 to x

You can also do the declaration and binding in just ONE line of code

1
let x = 5;

Now when we access the x, we get the value 5

2
console.log(x);
