JavaScript Conditionals

What are Conditionals?

Conditionals in JavaScript allow you to make decisions in your code based on whether a certain condition is true or false. Using conditionals, you can execute different blocks of code depending on the result of these conditions.

The if Statement

The most basic form of conditional is the if statement. It checks if a condition is true, and if it is, it executes the code inside the block.

1
let age = 18;
if (age >= 18) {
console.log("You are an adult.");
}

In this example, if age is greater than or equal to 18, the message "You are an adult." will be displayed in the console.

The else Statement

Sometimes, you want to do something if a condition is false. You can use the else statement to specify what to do when the if condition is false.

2
let age = 16;
if (age >= 18) {
console.log("You are an adult.");
} else {
console.log("You are a minor.");
}

In this example, since age is less than 18, the message "You are a minor." will be displayed in the console.

The else if Statement

If you need to check multiple conditions, you can use the else if statement. This allows you to test more than two conditions in a sequence.

3
let age = 70;
if (age < 18) {
console.log("You are a minor.");
} else if (age >= 18 && age <= 65) {
console.log("You are an adult.");
} else {
console.log("You are a senior.");
}

Here, the condition checks for three cases: age less than 18, between 18 and 65, and over 65. Since age is 70, the message "You are a senior." will be displayed.

Logical Operators in Conditionals

You can combine multiple conditions in an if, else if, or else block using logical operators.

The && (AND) Operator

The && operator checks if both conditions are true:

4
let age = 25;
let hasPermission = true;
if (age >= 18 && hasPermission) {
console.log("You can access this area.");
} else {
console.log("Access denied.");
}

This will check if the person is both over 18 and has permission. If both conditions are true, the message "You can access this area." will be displayed.

The || (OR) Operator

The || operator checks if at least one condition is true:

5
let age = 16;
let hasPermission = true;
if (age >= 18 || hasPermission) {
console.log("You can access this area.");
} else {
console.log("Access denied.");
}

Since hasPermission is true, the message "You can access this area." will be displayed, even though age is less than 18.

Short-circuiting in Logical Operators

Logical operators in JavaScript also "short-circuit," meaning they stop evaluating as soon as the result is determined:

  • With &&, if the first condition is false, the second condition won’t be evaluated.
  • With ||, if the first condition is true, the second condition won’t be evaluated.

Switch Statement

The switch statement is an alternative to using multiple else if statements. It's used when you have many possible conditions based on the value of a single variable.

6
let day = 3;
switch (day) {
case 1:
console.log("Monday");
break;
case 2:
console.log("Tuesday");
break;
case 3:
console.log("Wednesday");
break;
default:
console.log("Invalid day");
}

In this example, the value of day is 3, so the message "Wednesday" will be displayed. If day was anything besides 1, 2, or 3, the default case would be executed.

Summary of Key Points

  • The if statement allows you to test conditions and execute code accordingly.
  • The else and else if statements help you handle multiple conditions.
  • Logical operators like && (AND) and || (OR) let you combine multiple conditions.
  • The switch statement provides an alternative way to handle multiple conditions based on one variable.
Conditionals are essential for controlling the flow of your JavaScript program based on specific conditions!
