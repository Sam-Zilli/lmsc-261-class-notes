Data Types Overview

What are they?

Variables can hold different types of data. The type of data determines what operations can be performed on it.

You can find out what data type a variable is holding by using typeof()

1
2
let x = 5;
console.log(typeof(x))

number

Numbers

MDN Number Documentation

Integers (whole numbers like 1,2,3,4,-35) and Floating Point (numbers with decimal places like 3.1415)

Numbers are written without quotes around them, like this...

1
let x = 5;

Operations

Numbers can be added, subtracted, multiplied, divided, and more...

1
2

let y = 15;

console.log(y - x);

Strings represent text and are written inside single quotes or double quotes.

Operations

You can use operators on Strings, but they behave differently than Numbers.

Booleans

MDN Boolean Documentation

Booleans represent a truth value and can either be true or false.

Booleans are commonly used in conditional statements:

1
2
3
4
let isRaining = true;
if (isRaining) {
console.log("Take an umbrella!");
}

Arrays

MDN Array Documentation

Arrays are used to store multiple values in a single variable. An array is an ordered list of elements enclosed in square brackets [], with elements separated by commas.


Each element (item) in the Array does NOT need to be the same data type.


1
2
3
4
let fruits = ["apple", "banana", "cherry"];
console.log(fruits[0]);
console.log(fruits[1]);
console.log(fruits[2]);

apple

banana

cherry

You can access individual elements of an array using their index (starting from 0), as shown in the example above.
