Expressions

What are Expressions?

A fragment of code that produces a value.

1. Types of Expressions

Expressions can be classified into various types based on their complexity and function. Here are the most common types:

Literal Expressions

A literal expression is a value directly used in the program. This can be a number, string, or other fixed values.

1
42
"Hello, World!"

In this case, 42 and "Hello, World!" are literal expressions.

Arithmetic Expressions

Arithmetic expressions are used to perform mathematical calculations. They combine numbers and operators to produce a result.

2
5 + 3
10 * (2 + 3)

The first example, 5 + 3, evaluates to 8, while the second one, 10 * (2 + 3), evaluates to 50.

Logical Expressions

Logical expressions are used to perform logical operations. They return a Boolean value, either true or false.

3
true && false
5 > 3

The first example, true && false, evaluates to false, while 5 > 3 evaluates to true.

2. How Are Expressions Used in Code?

Expressions are a key part of programming because they are used in virtually every statement and expression in code. For example, expressions are often used in assignments, conditions, and loops.

In Assignments

When assigning a value to a variable, the value is an expression that gets evaluated and then assigned to the variable.

4
let x = 5 + 3;

The expression 5 + 3 is evaluated first, and then 8 is assigned to the variable x.

In Conditions

Expressions are often used in conditional statements to decide which block of code to execute.

5
if (5 > 3) {
console.log("True!");
}

The condition 5 > 3 is an expression that evaluates to true, so the code inside the if block will execute, printing "True!" to the console.

3. Expressions in Functions

In functions, expressions can be used to return values, perform calculations, or even control the flow of execution.

6
function add(a, b) {
return a + b;
}

Here, a + b is an expression that is evaluated and returned by the function add.

Quick Recap:

Literal Expressions: Fixed values like numbers or strings

Arithmetic Expressions: Perform mathematical calculations

Logical Expressions: Perform logical operations that result in Boolean values

Expressions in Code: Used in assignments, conditions, and functions to evaluate values
