JavaScript Functions

What is a Function?

A function in JavaScript is a block of reusable code that performs a specific task. Functions can take inputs, called parameters, and return an output, though they don’t always have to.

Declaring a Function

There are multiple ways to declare a function in JavaScript. Let’s start with the most common one — using the function keyword:

1
function greet() {
console.log("Hello, World!");
}

In this example, we define a function called greet. It doesn't take any parameters and simply logs "Hello, World!" to the console.

Calling a Function

To run the code inside a function, we call it by its name, followed by parentheses:

2
greet();

When you call greet(), it will execute the code inside the function and output "Hello, World!" to the console.

Functions with Parameters

Functions can accept inputs, called parameters, which can be used within the function. Here's an example of a function that takes a parameter:

3
function greet(name) {
console.log("Hello, " + name + "!");
}

In this case, the function greet takes a name parameter, and when called, it will output a personalized greeting:

4
greet("Alice");

The output would be: Hello, Alice!

Functions with Return Values

Functions can also return a value. This is useful when you want the function to calculate something and give you a result to use elsewhere in your code.

5
function add(a, b) {
return a + b;
}

The function add takes two parameters, a and b, and returns their sum. Here's how you can use it:

6
let result = add(3, 4);
console.log(result); // 7

The output would be: 7, which is the result of adding 3 and 4.

Summary of Key Points

  • A function is a block of reusable code.
  • Functions can accept parameters and return values.
  • You can define a function using the function keyword or an arrow function.
  • Functions can be anonymous and passed as arguments to other functions.
Now you know how to create and use functions in JavaScript! Functions are a powerful way to organize and reuse your code.
