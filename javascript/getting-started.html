Getting Started with JavaScript

Installing Node.js with Homebrew

Node.js is a runtime that allows you to run JavaScript outside the browser. It is essential for server-side JavaScript and many modern web development tools. Here's how to install it using Homebrew on macOS:

  • First, if you don't have Homebrew installed, open your terminal and run:
    • 1
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  • Once Homebrew is installed, update it with the following command:
    • 2
    brew update
  • Next, install Node.js by running:
    • 3
    brew install node
  • Once installed, verify the installation by checking the versions of Node.js and npm (Node Package Manager) with:
    • 4
    node -v
    npm -v

With Node.js installed, you can now run JavaScript files outside of the browser, use npm to manage packages, and build more advanced projects!

What is JavaScript?

JavaScript is a programming language that allows you to create interactive effects within web browsers. It is an essential part of web development alongside HTML and CSS.

Where Do You Write JavaScript?

JavaScript code is typically written in .js files, which stand for JavaScript. These files contain the code that adds interactivity to websites.

Creating Your First JavaScript File

To create a simple JavaScript file, follow these steps:

  • Create a new file and name it script.js (or any name ending in .js).
  • Write your JavaScript code inside the file. For example:
1
console.log("Hello, world!");

This is a simple JavaScript command that logs the message "Hello, world!" to the console.

Linking JavaScript to Your HTML

To use your JavaScript in an HTML file, you need to link the .js file to your HTML. You do this by including a <script> tag in the HTML file. Here's an example:

1
<!DOCTYPE html>
<html>
<head>
<title>My First JavaScript</title>
</head>
<body>
<h1>Welcome to JavaScript</h1>
<script src="script.js"></script>
</body>
</html>

In the above example, the <script src="script.js"> tag links the script.js JavaScript file to the HTML file. When you open this HTML file in a web browser, the JavaScript will run, and you'll see "Hello, world!" in the browser's console.

Running Your JavaScript Code

Once you have linked your JavaScript file to an HTML file, you can open the HTML file in a browser and use the browser's developer tools (usually accessible by pressing F12) to view the output of your JavaScript code in the console.

Summary of Key Points

  • JavaScript files end in .js.
  • You can write JavaScript code in any text editor and save the file with the .js extension.
  • To use JavaScript in an HTML file, link it using the <script> tag.
  • Use console.log() to output messages to the browser's console for debugging.
  • Use Chrome's "Inspect" tool to view the code of other websites and see how JavaScript is being used in real projects.
  • Node.js can be installed with Homebrew using the command brew install node.

Next Steps

Once you are comfortable with the basics, you can move on to more advanced topics like variables, functions, and loops. JavaScript is a powerful language, and learning it opens up a lot of possibilities for creating dynamic websites and applications.
