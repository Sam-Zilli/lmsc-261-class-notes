JavaScript Loops

What are Loops?

Loops in JavaScript allow you to repeatedly execute a block of code as long as a specified condition is true. They are useful for tasks like iterating over arrays or repeating actions multiple times without writing redundant code.

Types of Loops in JavaScript

JavaScript has several types of loops, including:

  • while loop
  • do...while loop
  • for loop

The while Loop

The while loop will repeatedly execute a block of code as long as the specified condition evaluates to true.

1
let i = 0;
while (i < 5) {
console.log(i);
i++;
}

In this example, the while loop will run as long as i is less than 5. After each iteration, i is incremented by 1. The output will be:

  • 0
  • 1
  • 2
  • 3
  • 4

The do...while Loop

The do...while loop is similar to the while loop, but the condition is checked after the code inside the loop is executed, ensuring that the code is always run at least once.

2
let i = 0;
do {
console.log(i);
i++;
} while (i < 5);

Here, the do...while loop will execute the block of code at least once, even if the condition is false initially. The output will be the same as the while loop example:

  • 0
  • 1
  • 2
  • 3
  • 4

The for Loop

The for loop is used when you know beforehand how many times you want to run the loop. It consists of three parts: initialization, condition, and iteration.

3
for (let i = 0; i < 5; i++) {
console.log(i);
}

The for loop consists of three main parts:

  • Initialization: let i = 0 - This sets up the loop counter.
  • Condition: i < 5 - The loop will continue as long as this condition is true.
  • Iteration: i++ - This increments the counter after each iteration.

The output will be the same as the previous loops:

  • 0
  • 1
  • 2
  • 3
  • 4

Breaking and Continuing Loops

You can control the flow of loops using the break and continue statements:

The break Statement

The break statement immediately exits a loop when a specified condition is met:

5
for (let i = 0; i < 10; i++) {
if (i === 5) {
break;
}
console.log(i);
}

Here, the loop will stop when i reaches 5, so the output will be:

  • 0
  • 1
  • 2
  • 3
  • 4

The continue Statement

The continue statement skips the current iteration and moves on to the next one:

6
for (let i = 0; i < 5; i++) {
if (i === 2) {
continue;
}
console.log(i);
}

In this example, when i is equal to 2, the loop skips that iteration. The output will be:

  • 0
  • 1
  • 3
  • 4

Summary of Key Points

  • The while loop runs as long as the condition is true.
  • The do...while loop runs at least once, even if the condition is false.
  • The for loop is used when you know how many times you want to repeat the code.
  • Loops can be used to iterate over arrays and other data structures.
  • You can control the flow of a loop with the break and continue statements.
Loops are a powerful tool for repeating code and iterating over data in JavaScript!
