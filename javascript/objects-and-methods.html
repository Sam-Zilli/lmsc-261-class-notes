Objects and Methods in JavaScript

What Are Objects in JavaScript?

In JavaScript, an object is a collection of properties and methods. Properties are values associated with the object, while methods are functions that belong to the object. Objects are used to store and manage data in a structured way.

Creating an Object

Objects can be created using curly braces ({}) with key-value pairs. Each key is called a property, and the value can be anything, such as a string, number, array, or even another object.

1
let person = {
name: "John",
age: 30,
city: "New York"
};

In this example, the person object has three properties: name, age, and city.

Accessing Object Properties

You can access the properties of an object using either dot notation or bracket notation:

2
console.log(person.name); // Outputs: "John"
console.log(person["age"]); // Outputs: 30

In the example above, person.name and person["age"] are used to access the properties of the person object. Both methods work equally well, but dot notation is more commonly used.

What Are Methods?

Methods are functions that are part of an object. They allow objects to perform actions or manipulate their own data. You can define a method inside an object by assigning a function to a property.

3
let person = {
name: "John",
age: 30,
city: "New York",
greet: function() {
}
};

Here, the greet method is defined inside the person object. When called, it will log a greeting message that includes the name property of the object using the this keyword.

Calling Methods

You can call an object's method using dot notation:

4
person.greet(); // Outputs: "Hello, my name is John"

In this case, calling person.greet() invokes the method, and it prints the greeting message to the console.

Console as an Object with Methods

An example of an object with methods that you are likely already familiar with is console. The console object is built into JavaScript and has several methods, including console.log(), which is commonly used to display messages in the console.

5
console.log("This is a log message!");
console.log("Another message with a value:", 42);

In this example, console.log is a method of the console object. You can use it to log messages or other values to the browser's developer tools console.

Summary of Key Points

  • Objects are collections of properties (key-value pairs) and methods (functions associated with the object).
  • Methods are functions defined inside an object that allow the object to perform actions or manipulate its properties.
  • You can access object properties using dot notation or bracket notation.
  • The console object in JavaScript contains several useful methods, such as console.log() for logging messages to the console.

Next Steps

Now that you've learned the basics of objects and methods, you can explore more advanced topics like object inheritance, the this keyword, and how to manipulate data within objects. Understanding how to work with objects is fundamental to writing more complex and organized JavaScript code.
