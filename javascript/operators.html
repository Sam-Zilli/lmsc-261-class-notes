JavaScript Operators

What Are Operators?

Operators in JavaScript are special symbols used to perform operations on variables and values. They are fundamental to JavaScript programming and can be used to perform arithmetic, compare values, assign values, and more.


Arithmetic Operators

Arithmetic operators are used to perform basic mathematical operations on numbers:

  • + (Addition)
  • - (Subtraction)
  • * (Multiplication)
  • / (Division)
  • % (Modulo - Remainder)
  • ++ (Increment)
  • -- (Decrement)
Example:

let a = 5;
let b = 2;
console.log(a + b); // 7
console.log(a - b); // 3
console.log(a * b); // 10
console.log(a / b); // 2.5
console.log(a % b); // 1

Assignment Operators

Assignment operators are used to assign values to variables. The most common assignment operator is the equals sign (=), but there are also compound assignment operators for shorthand operations:

  • = (Simple Assignment)
  • += (Addition Assignment)
  • -= (Subtraction Assignment)
  • *= (Multiplication Assignment)
  • /= (Division Assignment)
  • %= (Modulo Assignment)
Example:
1
2
3
4
5
let x = 5;
x += 3; // x = x + 3;
console.log(x); // 8
x *= 2; // x = x * 2;
console.log(x); // 16

Assignment vs. Equality

It's important to distinguish between the assignment operator (=) and equality operators (== and ===), as they may look similar but serve different purposes.

The Assignment Operator (=)

The = operator is used to assign a value to a variable:

1
2
let x = 5;
console.log(x); // 5

In this example, the value 5 is assigned to the variable x using the assignment operator =.

The Equality Operators (==) and (===)

Equality operators are used to compare two values. There are two main equality operators:

  • == (Loose Equality): Compares two values for equality, performing type coercion if needed.
  • === (Strict Equality): Compares both the value and the type, without performing type coercion.
Loose Equality (==):

The == operator compares two values and performs type coercion if necessary:

1
2
3
let a = "5";
let b = 5;
console.log(a == b); // true

Here, the string "5" is coerced into the number 5, so the comparison is true.

Strict Equality (===):

The === operator does not perform type coercion and checks both the value and the type:

5
let a = "5";
let b = 5;
console.log(a === b); // false

In this case, the comparison is false because the types are different: one is a string and the other is a number.


Comparison Operators

Comparison operators are used to compare two values and return a boolean value (true or false). Common comparison operators include:

  • == (Equal to)
  • != (Not equal to)
  • > (Greater than)
  • < (Less than)
  • >= (Greater than or equal to)
  • <= (Less than or equal to)
Example:
1
2
3
4
5
let x = 10;
let y = 20;
console.log(x == y); // false
console.log(x < y); // true
console.log(x >= y); // false

Logical Operators

Logical operators are used to combine multiple conditions:

  • && (AND)
  • || (OR)
  • ! (NOT)
Example:

3
4

let a = true;
let b = false;
console.log(a && b); // false
console.log(a || b); // true
console.log(!a); // false

Summary of Operators

  • Arithmetic operators perform mathematical operations.
  • Assignment operators assign values to variables. Be careful with = for assignment vs. == and === for equality comparisons.
  • Comparison operators are used to compare values and return a boolean result.
  • Logical operators combine conditions for decision making.
