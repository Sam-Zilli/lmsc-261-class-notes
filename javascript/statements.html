Statements

What are Statements?

A statement is a complete instruction in a program that performs an action. Unlike expressions, statements do not return a value but instead control the flow of the program.

1. Types of Statements

Statements can be classified based on their functionality and the actions they perform. Below are the most common types:

Declaration Statements

A declaration statement is used to declare variables or constants. These statements define the variable and optionally assign it a value.

1
let x = 10;
const PI = 3.14;

In this case, let x = 10; declares a variable x and assigns it the value 10, while const PI = 3.14; declares a constant PI with a value of 3.14.

Conditional Statements

Conditional statements allow the execution of code based on whether a condition evaluates to true or false. These statements control the flow of execution.

2
if (x > 5) {
console.log("x is greater than 5");
} else {
console.log("x is less than or equal to 5");
}

The if statement checks if the value of x is greater than 5 and executes the corresponding code block based on the result.

Loop Statements

Loop statements repeatedly execute a block of code as long as a condition is true. These are used to automate repetitive tasks.

3
for (let i = 0; i < 5; i++) {
console.log(i);
}

The for loop here runs 5 times, printing the values of i from 0 to 4 to the console.

2. How Are Statements Used in Code?

Statements control the flow of the program and are integral to performing tasks. They are used in combination with expressions to execute actions in the program.

In Assignments

Statements are used to assign values to variables or constants. These assignments are usually simple expressions that evaluate to a value and then get assigned to a variable.

4
let result = 10 + 5;

The statement let result = 10 + 5; assigns the value 15 to the variable result.

In Loops

Loop statements execute a block of code multiple times, making them useful for repetitive tasks, such as iterating through arrays or performing calculations.

5
while (x > 0) {
x--;
console.log(x);
}

The while loop continues to execute as long as x is greater than 0, decrementing its value and printing it at each iteration.

3. Control Flow with Statements

Statements are the foundation of controlling the program flow. Without them, the program wouldn't know which blocks of code to execute under different conditions.

6
if (x % 2 === 0) {
console.log("x is even");
} else {
console.log("x is odd");
}

This if-else statement checks if x is even or odd and prints the result to the console.

Quick Recap:

Declaration Statements: Declare variables and constants

Conditional Statements: Control the flow of execution based on conditions

Loop Statements: Repeatedly execute code while a condition is true

Control Flow: Statements determine the flow of execution in the program
