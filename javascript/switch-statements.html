JavaScript Switch Statements

What is a Switch Statement?

A switch statement in JavaScript is a control flow statement that allows you to execute one of many blocks of code based on the value of a variable or expression. It’s often used as a more readable alternative to multiple if-else conditions when you have many possible conditions to check.

Syntax of a Switch Statement

The syntax of a switch statement consists of the switch keyword, followed by an expression to evaluate, and then one or more case labels. The statement checks the expression against each case, and if a match is found, the corresponding block of code is executed.

1
let fruit = "apple";
switch (fruit) {
case "apple":
console.log("This is an apple!");
break;
case "banana":
console.log("This is a banana!");
break;
default:
console.log("Unknown fruit!");
}

In this example, the value of fruit is checked against the case labels. If the value matches "apple", it will log This is an apple! to the console. If it doesn't match any of the cases, the default block will execute.

Understanding the Break Statement

The break statement is used to terminate the switch block and prevent the code from continuing to check the remaining case labels. Without break, the code would "fall through" to the next case and execute it, even if it doesn’t match the expression.

2
let color = "red";
switch (color) {
case "red":
console.log("Red is a color of passion.");
case "blue":
console.log("Blue is the color of the sky.");
case "green":
console.log("Green is the color of nature.");
}

If you run the above code without break, it will log all three messages regardless of the value of color. For proper behavior, you should add break after each case block to stop the flow after a match is found.

Using the Default Case

The default case is an optional final case that runs if no other case matches the expression. It acts like a fallback in case none of the specified values are found.

3
let animal = "dog";
switch (animal) {
case "cat":
console.log("It's a cat!");
break;
case "dog":
console.log("It's a dog!");
break;
default:
console.log("Unknown animal!");
}

In this example, the switch checks the value of animal. If it matches "dog", it will log It's a dog! to the console. If there’s no match, the default case will execute, logging Unknown animal!.

Switch with Expressions

You can also use expressions inside the switch statement. The expression doesn't have to be a simple variable or string—it can be any expression that returns a value.

4
let age = 25;
switch (true) {
case (age >= 18):
console.log("Adult");
break;
case (age < 18):
console.log("Minor");
break;
}

In this example, the switch statement checks the condition of age. Since the condition age >= 18 is true, it will output Adult.

Summary of Key Points

  • The switch statement checks an expression against multiple cases and executes the block of code for the first matching case.
  • The break statement is used to prevent "fall-through" behavior, where multiple cases execute one after another.
  • The default case acts as a fallback if no case matches the expression.
  • You can use expressions in the switch statement to check complex conditions.
