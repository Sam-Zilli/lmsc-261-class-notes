Variables

What are they?

Variables are containers for storing data values.

Declaring Variables

In this class, we will use the 'let' keyword to declare (make) variables.

You may see code in resources/online that use 'var' or 'const' instead of 'let'

1
let x;

Binding / Assigning Value to Variables

We use this syntax to put a value into a variable

2
x = 5;

The single equals sign is called the Assignmnt Operator

Visual Example:

1.) Declare (make) a variable called x

2.) Bind the value 5 to x.

You can also do the declaration and binding in just ONE line of code
1
let x = 5;
Now when we access the x, we get the value 5
2
console.log(x);
5

Now that the variable x exists, we can reassign it new values without using the 'let' keyword anymore
3
4
x = "Hello, World!";
console.log(x);
Hello, World!

Variables can hold any data type.
