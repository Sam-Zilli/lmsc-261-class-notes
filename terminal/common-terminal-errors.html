Common Terminal Errors

1. cd: no such file or directory - Directory Not Found

This error occurs when you try to navigate to a directory that doesn't exist.

Example:

$ cd folderThree
cd: no such file or directory: folderThree

Explanation: The error occurs because the directory folderThree does not exist in the current path, so the system is unable to navigate to it.



Possible Fixes:

  1. Check the spelling of the directory you are trying to navigate to.
  2. Use the ls command to list the directories and files in the current location.
  3. Change to a different directory that exists.

2. zsh: command not found: - Command Not Found

This error occurs when you don't use a valid command.

Example:

$ example.js
zsh: command not found: example.js

Possible Fixes:

  1. Did you forget the command before the file name? For example: $ node example.js
  2. Check the spelling of the command. Find common commands here.

3. fatal: not a git repository - Not a Git Repo

This error occurs when you try to use a git command in a non-git directory

Example:

$ git add example.js
fatal: not a git repository (or any of the parent directories): .git

Possible Fixes:

  1. Are you in a git repo? Run $ ls -ato check if there is a .git file
