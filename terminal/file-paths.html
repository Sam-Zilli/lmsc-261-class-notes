UNDERSTANDING FILE PATHS

What is a File Path?

A file path is the location of a file or directory on your computer. It shows the hierarchy of directories (folders) leading to a specific file.

In the terminal, file paths are used to navigate to different files and directories.

File paths can be absolute or relative:

  • Absolute Path: An absolute path starts from the root directory and specifies the complete path to a file or directory. It is the full path to the file or directory from the root.

  • Relative Path: A relative path specifies the location of a file or directory in relation to the current directory.

Examples of File Paths

Absolute Path

An absolute path gives the complete address of a file starting from the root directory. It does not depend on the current working directory.

Example: The absolute path to file.js no matter where you are would look like this:

/Users/samz/Documents/intro-to-programming/file.js


Relative Path

A relative path is based on the current directory. It does not start from the root but from the current directory where you are located.

Example: If you are currently in the /Users/samz/Documents directory and you want to access a file.js the relative path would be:

intro-to-programming/file.js
