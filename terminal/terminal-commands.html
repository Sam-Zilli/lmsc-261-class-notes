TERMINAL COMMANDS

What are Terminal Commands?

Terminal commands are used to interact with your computer's operating system through a command-line interface (CLI). By typing specific commands, you can perform various tasks like navigating directories, managing files, and running scripts.

Common Terminal Commands

1. pwd - Print Working Directory

The pwd command displays the absolute path to the current working directory. It stands for "print working directory."

$ pwd

/home/user/Documents

2. ls - List Directory Contents

The ls command lists the files and directories in the current directory.

$ ls

file1.txt

file2.txt

Desktop

Documents

Applications

3. cd - Change Directory

The cd command is used to change the current working directory.

$ cd Documents

$ ls

Music

homework.pdf

cd .. moves you to a level UP in the tree.

$ cd ..

cd ~ moves you to the home directory.

$ cd ~

4. mkdir - Make Directory

The mkdir command creates a new directory.

$ mkdir intro-to-programming

$ ls

Pictures

homework.pdf

intro-to-programming

5. rm - Remove File

The rm command removes a file or directory.

$ rm homework.pdf

$ ls

Pictures

intro-to-programming

6. touch - Create a New File

The touch command is used to create a new, empty file.

$ touch coding-homework.js

$ ls

Pictures

intro-to-programming

coding-homework.js

7. mv - Renaming OR Moving a file or directory

The mv command does both renaming and moving depending on what you put after it.


To RENAME a file called coding-homework.js to be called new-name.js:

$ mv coding-homework.js new-name.js

$ ls

Pictures

intro-to-programming

new-name.js

To MOVE a file called new-file.js into the directory intro-to-programming

$ mv new-name.js intro-to-programming

$ ls

Pictures

intro-to-programming

$ cd intro-to-programming

$ ls

new-name.js
